# 03: Designated Responsibility

Specific people are responsible for specific things. A person might be responsible for reporting on a particular area, managing a set of locations, or covering a defined group.

When something needs to be done or reported, it's clear who is accountable. If it doesn't get done, the responsible person is identifiable.

This results in:

* A clear link between people and the things they're responsible for
* The ability to tell who should have done something and whether they did
* Accountability that traces back to named individuals
