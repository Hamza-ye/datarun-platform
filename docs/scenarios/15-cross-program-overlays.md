# 15: The Same Work Serving Multiple Audiences

A single reporting or data collection effort produces information that multiple independent stakeholders care about. Each stakeholder has their own perspective: different criteria for what counts as complete, different timelines for when it's due, different ways they want to see the data summarized.

A health ministry may want results grouped by facility type. A finance ministry may want the same data cut by commodity cost. A donor may want geographic coverage. All are looking at the same underlying work, but through different lenses.

If one stakeholder's view has gaps, that shouldn't block another stakeholder from acting on what's already available. Yet when inconsistencies arise, it should be possible to trace them back to the common source.

This results in:

* Multiple perspectives on the same underlying work
* Each stakeholder seeing what matters to them, on their own timeline
* Traceability from any stakeholder's view back to the original records
