# 10: Work That Depends on Changing Conditions

Not everything is planned upfront. Sometimes, the need for action arises from what's already been observed. If a certain condition is met — a threshold crossed, a pattern detected, a gap identified — then something needs to happen in response.

As conditions change, the set of things that need attention changes too. Things that previously needed action may no longer require it. New things may start needing attention. What was done in the past may influence what needs to happen next.

This results in:

* Work that emerges from observed conditions rather than being planned in advance
* A shifting set of things requiring attention based on what's currently known
* A feedback relationship where previous actions influence future needs
